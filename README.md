# MusixApp-FullStack
🎵 MusixApp - Full-Stack Music Streaming Platform A modern, responsive music streaming application built with a focus on seamless user experience and robust state management.

# Tech Stack
1. Frontend Library : React or HTML5/Vanilla JS (using standard Bootstrap classes to layout the grids for the song cards).
2. Backend : Java 17 & Spring Boot as the core application framework.
3. Database : MySql
